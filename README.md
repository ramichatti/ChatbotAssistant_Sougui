## 📌 Overview

**Chatbot Assistant Sougui** is an AI-powered chatbot built using a **local LLM (Qwen 5.1 via Ollama)**.  
It connects to the **Sougui_DWH Data Warehouse** to help users interact with commercial data using **natural language**.

It simplifies decision-making by allowing users to query:
- Sales 📊
- Purchases 🛒
- Inventory 📦
- Business insights 📈

---

## 🚀 Features

- 💬 Natural language chatbot interface  
- 📊 Sales & business data analysis  
- 🔎 Automatic SQL query generation  
- 📈 KPI & reporting support  
- ⚡ Local AI processing (no cloud dependency)  
- 🔐 Secure access to internal data warehouse  

---

## 🧠 Tech Stack

- **LLM**: Qwen 5.1  
- **AI Runtime**: Ollama  
- **Backend**: Python / Node.js  
- **Database**: Sougui_DWH (Data Warehouse)  
- **BI Layer**: Analytical queries & reporting
